# 問題
回答idとして自然数aが、N個与えられます。
回答にはそれぞれ、タグが付与されます。
タグは、タググループに属しています。
タグを選択して回答を絞り込むツリー構造のJSON文字列を生成してください。


## 成約

```
N<=100
```

## フォーマット

```
N
a1 a2 a3 a4
t t t
t t t t t
t t t t t t
t t
tg t t t t
tg t t t t
tg t t t t
```

